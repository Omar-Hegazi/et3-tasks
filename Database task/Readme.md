## How to run
- Install SQL SERVER 2019
- Install SQL SERVER MANAGEMENT STUDIO (Optional)
- Open sqlserver.txt in SQL SERVER terminal or SQL SERVER MANAGEMENT STUDIO
- Run sqlserver.txt & Done
